Laboratory Work №3: Class Diagrams
This section presents Class Diagrams that model the static structure and relationships within the selected systems.

1. Car Insurance System
Core Classes: InsuranceCompany, Customer, Policy, Vehicle, and Claim.  
Relationships: The company manages policies that cover specific vehicles and are linked to customer claims.  

2. ATM System
Core Classes: Bank, ATM, Account, SavingsAccount, and Transaction.  
Relationships: The Bank manages accounts and ATMs, which execute transactions and dispense cash.  

3. Medical Clinic System
Core Classes: Person (Base Class), Patient, Doctor, Appointment, and MedicalRecord.  
Relationships: Doctors conduct appointments with patients, who own encrypted medical records.
