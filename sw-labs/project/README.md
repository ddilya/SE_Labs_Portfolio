# Task Manager

## Project Idea
Task Manager is an application designed to help users organize, manage and track their tasks.

## Problem Statement
People often forget deadlines and struggle to organize their work. This application helps improve productivity.

## Objectives
- Organize tasks
- Track progress
- Manage deadlines

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Set deadlines
- Mark tasks as completed
- Set task priorities

## Technologies
Frontend: HTML/CSS/JavaScript  
Backend: Python / Java  
Database: MySQL
